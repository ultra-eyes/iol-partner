# Vivity vs PureSee：EDOF IOL比較リファレンス

> **対象**：視能訓練士・眼科医療従事者
> **最終更新**：2026年5月
> **用途**：多焦点IOL選定カウンセリング・院内教育・コンテンツ作成基礎資料

---

## TL;DR（結論サマリー）

| 評価項目 | 優位 | エビデンス強度 |
|---|---|---|
| 自覚的ハロー・グレア（dysphotopsia） | **Vivity** | ★★★（直接比較あり） |
| 高空間周波数コントラスト感度 | **PureSee** | ★★★（直接比較あり） |
| 中間視（60–70cm） | **Vivity** | ★★★ |
| 近見（40–50cm） | **PureSee**（傾向） | ★★（NS） |
| 瞳孔径への寛容性 | **PureSee** | ★★ |
| 偏心への寛容性 | **PureSee** | ★★ |
| 遠方視（far VA） | comparable | — |

**「Vivity = コントラスト・ハローに強い」は半分正解、半分不正確。** ハローグレアではVivity優位だが、コントラスト感度（特に高空間周波数）では純粋屈折型のPureSeeが優位という直接比較結果が出ている。

---

## 1. ハローグレア：Vivity優位

### Lin et al.（Clinical Ophthalmology, 2025）retrospective head-to-head
2ヶ月術後の自覚症状評価：

| 症状 | PureSee | Vivity |
|---|---|---|
| グレア「always/often」 | 18.0% | 2.6% |
| ハロー「frequently」 | 12.8% | 0% |

### 注意点：ベンチ vs 自覚症状の乖離
Nature Scientific Reports（2025）の光学ベンチ評価では、両モデルが**ほぼ同一のハロー様パターン**を生成。光学的特性と自覚症状には乖離があり、自覚症状ベースではVivity優位。Neural adaptationの寄与も示唆。

---

## 2. コントラスト感度：PureSee優位（特に高空間周波数）

### Lin et al. head-to-head data

| 条件 | 結果 | p値 |
|---|---|---|
| Photopic 12 cpd | PureSee有意に高い | 0.012 |
| Photopic 18 cpd | PureSee有意に高い | 0.012 |
| Mesopic 18 cpd | PureSee有意に高い | 0.038 |

低〜中空間周波数では両者comparable、**高空間周波数（細かいディテール）でPureSeeが明確に優位**。

### メカニズム

| レンズ | 光学設計 | 結果 |
|---|---|---|
| **PureSee** | Continuous Power Technology（純粋屈折型）。前面非球面＋後面の連続的屈折力変化 | 単焦点様の光学品質を保持。波面整合性が高い |
| **Vivity** | X-WAVE Wavefront-shaping。後面の2つのsmooth surface transition（plateau ~1µm + curvature change at 2.2mm）で波面をストレッチ＆シフト | 波面の不連続点が存在。SAは中心4mm内で正負振動 |

---

## 3. 中間視・近見の defocus curve

### PureSee
- VA <0.20 logMAR を −1.6D まで維持（最適屈折で −1.9D まで延長可能）
- 等価調節幅：約 2.0D
- **40–50cm（近見）で優位の傾向**

### Vivity
- 等価調節幅：+2.00 〜 +2.50D
- 中間視 66cmで >20/25 のUDIVA
- **66–70cm（中間視）で有意に優位**（Lin et al. P=0.045）

## 4. 患者選定アルゴリズム

### Vivityが第一選択となる患者プロファイル
- 夜間運転が多い（営業職、配送、夜勤シフト）
- デスクワーク中心（PC作業60–70cm）
- 楽器演奏・ゴルフ等、中間距離が重要な趣味
- ハロー・グレアへの感受性が高い（既往の言及あり）
- 瞳孔径が3–4mmで安定している

### PureSeeが第一選択となる患者プロファイル
- 読書・手芸・新聞が日常の中心
- 暗所で瞳孔が大きく開く患者
- 色彩識別が重要な職業（デザイナー、画家、ネイリスト、写真家）
- スマートフォン使用頻度が高い（30cm）
- 単焦点経験者で「素直な見え方」を望む
- 角膜偏心や術後解剖学的リスクがある（centration tolerance）

### 注意したい相対禁忌
- **両者共通**：非現実的な期待、網膜疾患、矯正不能な角膜不整
- **Vivity**：強い瞳孔径感受性 → 散瞳しやすい患者要注意
- **PureSee**：「絶対に夜間ハローは嫌」と強く訴える患者

## 5. 患者カウンセリングアプローチ

### ヒアリングシート項目
1. **1日の活動分布**：朝の身支度、通勤、仕事、家事、夕方〜夜の過ごし方
2. **距離別作業時間**：30cm（手元）、60cm（PC・楽譜）、3m以上（運転・スポーツ）
3. **夜間活動**：運転頻度、屋外活動の有無
4. **譲れない見え方**：「これだけは絶対」を1つ選んでもらう
5. **過去の眼鏡経験**：単焦点・遠近両用・コンタクトの感想

### 1日のシーン別マッピング

| シーン | 距離 | 優位レンズ |
|---|---|---|
| 朝：新聞・スマホ・化粧 | 30–40cm | PureSee |
| 通勤：夜間運転 | 遠方 | Vivity |
| 通勤：スマホで地図 | 30cm | PureSee |
| 仕事：PC作業 | 60–70cm | Vivity |
| 仕事：書類・サイン | 30–40cm | PureSee |
| 仕事：会議・プレゼン | 中間〜遠方 | Vivity |
| 家事：料理（まな板・レシピ） | 40–60cm | comparable |
| 買い物：価格札・商品棚 | 中間 | Vivity |
| 夜：読書・編み物 | 30–40cm | PureSee |
| 夜：TV・映画 | 遠方 | comparable |
| 趣味：ゴルフ・楽器 | 中間〜遠方 | Vivity |
| 趣味：ネイル・色合わせ | 30cm + 色覚 | PureSee |
